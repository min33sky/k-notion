# K-notion

> Korean Notion

## Stack

- React
- Next.js
- Tailwindcss
- Convex
- Typescript

## Getting started

1. Install dependencies

```bash
pnpm install
```

2. DB (convex) start

```bash
pnpm convex dev
```

3. Dev Server start

```bash
pnpm dev
```
